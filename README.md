# ClothingSort

I decided to implement a sorting algorithm that would help ease the stress of choosing day to day outfits. This algorithm takes pieces of clothing and gives them a score based on weather, amount worn, and style, and it combines this to help generate the best possible outfit choices.

# Project Structure
clothing_item.py - defines the ClothingItem class
score.py - calculates an individual clothing item's score
coordination.py - evaluates compatability between 2 items
outfit_generator.py - generates and ranks outfit recommendations
test_cases.py - contains sample clothing items and test cases

# How to Run
1. Clone repository
   git clone https://github.com/lreyes2258/ClothingSort.git
2. Open project folder
3. Run test file
   python test_cases.py
